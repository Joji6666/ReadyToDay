### 📃 프로젝트 소개

- 오늘의 할 일을 기록하고 오늘의 날씨를 확인하세요! 오늘 하루를 준비할 수 있는 웹서비스입니다.
- 음성인식 기능을 이용하여 할 일을 기록할 수 있고 간단한  질문을 할 수 있는 챗봇이 준비되어있습니다.
